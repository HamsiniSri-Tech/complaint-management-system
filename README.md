# Complaint Management System

## Project Description
The Complaint Management System is a MERN Stack web application that allows users to register complaints online and track their status. Administrators can manage complaints, update their status, and resolve issues efficiently.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- User Registration
- User Login
- Submit Complaint
- View Complaint Status
- Admin Dashboard
- Update Complaint Status
- Delete Complaint
- Responsive UI

## Project Structure
```
client/
server/
Documentation/
```

## Installation

### Clone Repository
```bash
git clone https://github.com/HamsiniSri-Tech/complaint-management-system.git
```

### Backend
```bash
cd server
npm install
npm start
```

### Frontend
```bash
cd client
npm install
npm start
```

## Environment Variables

Create a `.env` file inside the `server` folder.

Example:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
